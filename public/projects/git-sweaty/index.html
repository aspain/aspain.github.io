What git-sweaty Does

This project is built for people who track workouts and want an interactive activity dashboard without building their own data pipeline. It supports both Strava and Garmin as data sources, fetches activity history, normalizes activity types, and publishes an interactive dashboard to GitHub Pages. The dashboard contains many clickable filter options, and hovering over the graphs displays tooltips with additional activity detail.

The quick start workflow supports a recommended setup mode that can create or reuse a GitHub fork, configure repository variables, and guide credentials for provider authorization. You can switch providers later, rerun setup, and optionally force a one-time full backfill.

The dashboard is 100% free to set up and run, and is hosted on GitHub Pages. It automatically refreshes daily via scheduled GitHub Actions. The dashboard is desktop and mobile friendly.

Quick Start Command

From the project README, this is the main one-command setup path for macOS and Linux:

bash <(curl -fsSL https://raw.githubusercontent.com/aspain/git-sweaty/main/scripts/bootstrap.sh)

During the initial setup, options include:

  • Data source (Strava or Garmin)
  • Unit preference (US or metric)
  • Week start (Sunday or Monday)
  • Add your Strava or Garmin profile link to the dashboard
  • Add direct links to activities in the graph tooltips
