git-sweaty

-

git-sweaty is for people who track workouts and want an interactive dashboard without building a custom data pipeline. It supports importing activities from Strava or Garmin, normalizes activity types, and publishes to GitHub Pages with intuitive filters and detailed hover tooltips.

Key features:

  • The quick-start flow can create or reuse an existing repository fork, configure repository variables, and guide provider authorization
  • You can switch between Strava or Garmin as the data source at any time
  • Works on desktop and mobile
  • Refreshes daily through scheduled GitHub Actions to pick up recent activities
  • Completely free: the dashboard is hosted on GitHub Pages

Quick Start Command

From the project README, this is the main one-command setup path for macOS and Linux:

bash <(curl -fsSL https://raw.githubusercontent.com/aspain/git-sweaty/main/scripts/bootstrap.sh)

During the initial setup, options include:

  • Setup mode:
    • Online-only - no local clone required, automatically creates or uses an existing fork
    • Advanced - use a local clone or fork + clone
  • GitHub Pages custom domain (if you have one, for example yoursite.example.com)
  • Data source (Strava or Garmin)
  • Unit preference (US or metric)
  • Week start (Sunday or Monday)
  • Add your Strava or Garmin profile link to the dashboard (Yes/No)
  • Add direct links to activities in the graph tooltips (Yes/No)
