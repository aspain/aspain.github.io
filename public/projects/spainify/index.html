What spainify Does

spainify runs as a set of locally hosted services on a Raspberry Pi to provide a single place for home media visibility and control. The now-playing interface renders artist, track title, and album artwork on an LCD screen, while the background color adapts dynamically from the artwork palette for a live ambient look.

Beyond display, the stack includes Sonos transport and grouping controls, weather dashboard views, and local APIs that support one-tap iOS shortcuts. A common pattern is adding the currently playing track directly to a Spotify playlist from a lock-screen shortcut, including duplicate protection.

  • Now-playing screen with dynamic color theming from album art.
  • Full Sonos controls: play, pause, skip, volume, room grouping, and presets.
  • OpenWeather-powered local forecast dashboard windows.
  • Display sleep and wake behavior tied to playback state and schedule.
  • Remote setup and redeploy commands to manage updates from your desktop.

Setup and Update Flow

The README documents a first-time setup path that clones the repo and runs a remote setup script over SSH. During setup, you configure service choices, OpenWeather credentials, and optional Spotify app credentials for media-actions endpoints.

git clone https://github.com/aspain/spainify.git && cd spainify && ./scripts/setup-remote.sh <pi-user>@<pi-ip> --fresh

After code changes, a standard redeploy command pulls latest commits on the Pi and restarts enabled services. This keeps the display and APIs in sync without a full reinstall.

ssh <pi-user>@<pi-host-or-ip> 'cd ~/spainify && git pull --ff-only && ./scripts/redeploy.sh'
